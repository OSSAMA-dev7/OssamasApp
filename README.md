# 🎤 Ossama's App - Speech to Text Converter

## 💙 Helping people who can't hear

A beautiful desktop application that converts speech to text in **English** and **Arabic**.

### ✨ Features

- 🎤 **Real-time Speech Recognition** - Convert speech to text instantly
- 🌍 **Multi-language Support** - English & Arabic (العربية)
- 💾 **Save Recordings** - Export conversations as text files
- 🎨 **Beautiful UI** - Modern, easy-to-use interface
- ⚡ **Fast & Responsive** - Lightweight and efficient
- 🔒 **Privacy** - Works offline, no data sent anywhere

### 📋 Requirements

- **Python 3.11+** (download from [python.org](https://www.python.org/downloads/))
- **Windows 10/11** or **Linux** or **Mac**
- **Microphone** (for speech input)
- **~2GB free space** (for voice models)
